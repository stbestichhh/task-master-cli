## Installation

Install task-master-cli with npm

```bash
  npm install @stbestichhh/task-master-cli@0.0.1
```

or [download latest release](https://github.com/stbestichhh/task-master-cli/releases/tag/v0.0.1)

```bash
  cd task-master-cli
  npm i -g .  
```
